HTML Basics


HTML stands for HyperText Markup Language. It is the standarf language used to create web pages

Features of HTML


  • Easy to learn and use
  • Platform-independent
  • Supports multimedia like images and videos

Steps to create a Basic HTML Page


  1. Open a text editor
  2. Write your HTML Code
  3. Save the file with a .html extention
  4. Open the file in a web browser
